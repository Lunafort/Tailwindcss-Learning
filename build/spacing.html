Padding

Utilities for controlling an element's padding. (inside selected element).p-{t|r|b|l}-{size}.example:

p-5

px-5

py-5

pt-5 pb-5

pl-5 pr-5

Margin

Utilities for controlling an element's margin. (outside selected element) .m-{t|r|b|l}-{size}.example:

m-5

mx-5

my-5

mt-5 mb-5

ml-5 mr-5

Space-Between

Utilities for controlling the space between child elements. (spacing between each child elements). .space-{x/y}-{size}.example:

  • space-x-{size}
    • 1
    2
    3
  • reversing children order
    • 1
    2
    3
  • vertical space between chidlren
    • 1
    2
    3
